安装 #

环境支持 #

由于是对element-plus的二次封装，同时基于vue3，因此不支持ie11

Edge ≥ 79Firefox ≥ 78Chrome ≥ 64Safari ≥ 12

本地安装 #

git clone https://github.com/SepVeneto/basic-comp.git

pnpm i

pnpm build:comps

# 或者通过npm link建立软连接

使用包管理器安装 #

# npm
npm i @sepveento/basic-comp

#yarn
yarn add @sepveento/basic-comp

#pnpm
pnpm i @sepveento/basic-comp
