输入框 ​

在element-plus的基础上，针对业务场景做了一些功能上的增强，同时通过透传保证了原生的正常用法。

基本用法 ​

对element-plus的原生属性做了最基本的透传，可以直接使用原生的属性

vue
<template>
  <bc-input v-model="text" />
</template>

<script lang="ts" setup>
import { ref } from 'vue'
defineOptions({
  name: 'Basic',
})
const text = ref('')
</script>

边框设置 ​

vue
<template>
  <bc-input v-model="text" width="183px" class="input" />
  <bc-input v-model="text" border="bottom" width="183px" class="input" />
  <bc-input v-model="text" border="none" width="183px" class="input" />
</template>

<script lang="ts">
import { defineComponent } from 'vue'
export default defineComponent({
  name: 'Border',
})
</script>

<script lang="ts" setup>
import { ref } from 'vue'
const text = ref('border-setting')
</script>

<style lang="scss" scoped>
.input:not(:last-child) {
  margin-right: 20px;
}
</style>
