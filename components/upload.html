Upload ​

上传弹框，自定义上传方式和下载模板

基本用法 ​

通过v-model可以控制上传弹框的显示与否，同时需要template-api和upload-api来实现模板下载和文件上传的具体实现。

vue
<template>
  <bc-button @click="visible = true">
    上传
  </bc-button>
  <bc-upload
    v-model="visible"
    :upload-api="upload"
    callback-type="file"
  />
</template>

<script lang="ts" setup>
import { ref } from 'vue'
defineOptions({
  name: 'Basic',
})
const visible = ref(false)
function upload(data: File) {
  // console.log(data.get('file'))
  console.log(data)
  return Promise.reject(new Error('error'))
}
</script>
