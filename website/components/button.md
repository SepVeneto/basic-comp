## 属性
| 属性           | 说明                                                      | 类型                | 可选值 | 默认值                  |
| :------------- | :-------------------------------------------------------- | :------------------ | :----- | :---------------------- |
| confirm | 是否需要操作确认提示 | boolean / string | - | false |
| tooltip | 是否以悬浮框展示按钮名称，主要用于按钮最小化后的提示 | boolean | - | false |
| mini | 是否最小化 | boolean | - | false |
